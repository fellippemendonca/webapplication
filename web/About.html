About this Software Application
This Software is being developed by the Quality Assurance Team of Cnova, willing to keep an updated repository of System API definitions, and use these knowledge for learning, validating and monitoring purposes.

Software Features:

  • API requests Search, Inclusion and Maintenance in a easy and fast access repository.
  • Rest and SOAP requests supported.
  • Dynamically generated requests filled in integration with Databank queries.
  • Response Validation Scenarios inclusion for any Request.
  • Request Scenarios Validation Schedule for environment monitoring purposes.
  • Detailed Log Visualization of Scheduled Executions.
  • Threaded massive execution of requests including "threadNumber" Parameter and the number of threads in it's value, to fire all requests at the same time.
  • Charts Generator. Automatically generate charts from queries included on the system.
